# 02 - Problemas Estructurales del Sistema Español

## Resumen Ejecutivo

Este módulo documenta los **problemas estructurales** del sistema político-administrativo español. Para cada problema se analiza:

- **Causa raíz**: Por qué existe el problema
- **Quién se beneficia**: Actores que tienen incentivos para mantenerlo
- **Impacto**: Consecuencias económicas y sociales
- **Por qué no se corrige**: Obstáculos para el cambio
- **Propuesta de solución**: Alternativas concretas
- **Riesgos de la solución**: Posibles efectos adversos

## Metodología de Análisis

Para cada problema aplicamos la siguiente estructura:

### 1. Descripción del Problema
- Qué es exactamente
- Cómo se manifiesta
- Datos cuantitativos disponibles

### 2. Análisis de Causas
```
Problema
    └── Causa inmediata
            └── Causa subyacente
                    └── Causa raíz (estructural)
```

### 3. Mapa de Intereses
- ¿Quién gana con el status quo?
- ¿Quién pierde?
- ¿Quién tiene poder para cambiarlo?
- ¿Por qué no lo hace?

### 4. Propuesta de Solución
- Medida concreta
- Mecanismo de implementación
- Recursos necesarios
- Timeline estimado

### 5. Análisis de Riesgos
- ¿Qué puede salir mal?
- ¿Cómo mitigarlo?
- ¿Hay efectos secundarios?

## Interdependencias entre Problemas

Muchos problemas están conectados:

| Problema A | → Alimenta → | Problema B |
|------------|--------------|------------|
| Partitocracia | → | Puertas giratorias |
| Puertas giratorias | → | Captura regulatoria |
| Captura regulatoria | → | Regulación anti-emprendimiento |
| Burocracia | → | Economía sumergida |
| Falta de transparencia | → | Corrupción |
| Corrupción | → | Desafección ciudadana |
| Desafección | → | Baja participación |
| Baja participación | → | Perpetuación de la partitocracia |

**Conclusión:** No basta con resolver problemas aislados. Se requiere una reforma estructural integral.
