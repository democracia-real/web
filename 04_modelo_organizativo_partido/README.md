# 04 - Modelo Organizativo del Partido

## Resumen Ejecutivo

Este módulo define la **estructura y funcionamiento interno** del nuevo partido político, diseñado para ser coherente con los principios que propugna: democracia interna real, transparencia radical, meritocracia y uso de tecnología.

> *"No podemos pedir a otros lo que no practicamos internamente."*

## Principios Organizativos

### 1. Coherencia Medios-Fines
El partido debe ser internamente lo que propone para el Estado:
- Separación de poderes internos
- Rendición de cuentas
- Meritocracia en nombramientos
- Transparencia total

### 2. Democracia Interna Real
No una democracia formal donde la cúpula controla todo:
- Primarias abiertas y competitivas
- Afiliados con voz y voto efectivo
- Órganos de control independientes
- Revocabilidad de cargos

### 3. Meritocracia sobre Lealtad
Los cargos se asignan por competencia demostrada:
- Currículum público de candidatos
- Evaluación de desempeño
- Rotación obligatoria
- Sin "familias" ni baronías

### 4. Transparencia Radical
Todo es público excepto datos personales protegidos:
- Finanzas en tiempo real
- Actas de reuniones
- Votaciones nominales
- Decisiones motivadas

### 5. Tecnología como Habilitador
La plataforma digital no es un añadido, es el centro:
- Participación online equiparada a presencial
- Votaciones verificables por blockchain
- Deliberación asíncrona
- Accesibilidad universal

## Estructura Orgánica Propuesta

```
AFILIADOS (base)
    ↓ eligen
ASAMBLEA GENERAL (máximo órgano)
    ↓ elige
CONSEJO CIUDADANO (dirección colegiada)
    ↓ designa
COMISIÓN EJECUTIVA (gestión diaria)

ÓRGANOS DE CONTROL (independientes):
- Comisión de Garantías
- Comisión de Ética
- Comisión Económica
```

### Órganos Principales

| Órgano | Función | Composición |
|--------|---------|-------------|
| **Asamblea General** | Máxima autoridad, aprueba líneas estratégicas | Todos los afiliados |
| **Consejo Ciudadano** | Dirección política, propuestas | Elegido por afiliados |
| **Comisión Ejecutiva** | Gestión operativa diaria | Elegida por Consejo |
| **Portavoz** | Representación pública | Elegido por afiliados |
| **Comisión de Garantías** | Resolución de conflictos, derechos | Independiente |
| **Comisión de Ética** | Cumplimiento código ético | Independiente |
| **Comisión Económica** | Auditoría financiera | Independiente |

### Separación de Poderes Interna

| Poder | Órgano | Función |
|-------|--------|---------|
| Legislativo | Asamblea + Consejo Ciudadano | Definir políticas |
| Ejecutivo | Comisión Ejecutiva | Implementar |
| Judicial | Comisión de Garantías | Resolver conflictos |
| Auditor | Comisión Económica + Ética | Controlar |

## Innovaciones Organizativas

### 1. Democracia Líquida Interna

| Mecanismo | Descripción |
|-----------|-------------|
| Voto directo | Cada afiliado vota en cada decisión |
| Delegación | Puede delegar en otro afiliado de confianza |
| Revocación | Puede retirar delegación en cualquier momento |
| Especialización | Delegar por áreas temáticas |
| Plataforma | Sistema verificable por blockchain |

### 2. Primarias Abiertas

| Elemento | Propuesta |
|----------|-----------|
| Participantes | Afiliados + simpatizantes registrados |
| Candidaturas | Requisitos mínimos de avales (bajos) |
| Debates | Obligatorios y públicos |
| Votación | Online y presencial, verificable |
| Resultados | Vinculantes, sin manipulación posterior |

### 3. Mandatos Limitados

| Cargo | Límite |
|-------|--------|
| Portavoz | 2 mandatos máximo |
| Consejo Ciudadano | 3 mandatos máximo |
| Comisión Ejecutiva | 2 mandatos máximo |
| Cargos electos | 3 legislaturas máximo |

### 4. Revocación de Cargos

| Mecanismo | Detalle |
|-----------|---------|
| Iniciativa | X% de afiliados del ámbito |
| Causas | Incumplimiento, conducta, pérdida confianza |
| Procedimiento | Consulta vinculante |
| Protección | Período mínimo de mandato |

### 5. Rotación y Límite Salarial

| Principio | Aplicación |
|-----------|------------|
| Rotación | Cargos ejecutivos rotan obligatoriamente |
| Tope salarial | Cargos públicos no superan X veces SMI |
| Devolución | Exceso se destina a causas sociales |
| Transparencia | Nóminas públicas |

## Afiliación y Participación

### Tipos de Vinculación

| Tipo | Derechos | Deberes |
|------|----------|---------|
| **Simpatizante** | Información, participar en primarias abiertas | Registro básico |
| **Afiliado** | Voto en todas las decisiones, candidaturas | Cuota (proporcional a ingresos) |
| **Colaborador** | Participar en grupos de trabajo | Compromiso de tiempo |

### Cuota de Afiliación

| Principio | Aplicación |
|-----------|------------|
| Proporcional | Según ingresos declarados |
| Voluntaria adicional | Quien quiera aportar más |
| Transparencia | Destino de fondos público |
| Mínimo simbólico | Para rentas bajas |

## Financiación del Partido

### Principios

| Principio | Aplicación |
|-----------|------------|
| Transparencia total | Ingresos y gastos públicos en tiempo real |
| Independencia | No aceptar donaciones que comprometan |
| Límites | Máximo por donante individual |
| Prohibiciones | Empresas con contratos públicos, sectores regulados |
| Crowdfunding | Campañas de microdonaciones |

### Fuentes de Financiación

| Fuente | Política |
|--------|----------|
| Cuotas de afiliados | Base principal |
| Subvenciones públicas | Aceptar con transparencia |
| Donaciones individuales | Límite por persona, publicidad |
| Crowdfunding | Para campañas específicas |
| Aportaciones de cargos | Porcentaje de salario público |

## Relación con Cargos Electos

### Compromisos

| Compromiso | Mecanismo |
|------------|-----------|
| Programa público | Compromisos medibles y verificables |
| Rendición de cuentas | Informes periódicos a afiliados |
| Accesibilidad | Canales de comunicación con ciudadanos |
| Coherencia | Voto según programa, no según cúpula |
| Aportación económica | Parte del salario al partido |

### Sanciones por Incumplimiento

| Incumplimiento | Consecuencia |
|----------------|--------------|
| Voto contrario al programa sin justificar | Expediente |
| Corrupción | Expulsión inmediata |
| Falta de transparencia | Sanción económica |
| Incumplimiento reiterado | Revocación de apoyo |

## Uso de Tecnología

### Plataforma Digital

| Función | Herramienta |
|---------|-------------|
| Votaciones | Sistema blockchain verificable |
| Deliberación | Foros estructurados |
| Propuestas | Plataforma de iniciativas |
| Transparencia | Dashboard en tiempo real |
| Comunicación | Canales directos |

### Inteligencia Artificial

| Aplicación | Uso |
|------------|-----|
| Síntesis de propuestas | Agrupar ideas similares |
| Detección de conflictos | Alertar sobre inconsistencias |
| Análisis de participación | Identificar sesgos |
| Matching de perfiles | Conectar afiliados con intereses comunes |

## Diferencias con Partidos Tradicionales

| Aspecto | Partido Tradicional | Este Partido |
|---------|---------------------|--------------|
| Toma de decisiones | Cúpula | Afiliados |
| Selección candidatos | Dedazo | Primarias abiertas |
| Transparencia | Mínima legal | Radical y proactiva |
| Mandatos | Ilimitados | Limitados |
| Rendición cuentas | Formal | Real y continua |
| Tecnología | Accesoria | Central |
| Ideología | Rígida | Pragmática, basada en evidencia |

## Riesgos y Mitigaciones

| Riesgo | Mitigación |
|--------|------------|
| Populismo interno | Deliberación informada, no solo votación |
| Baja participación | Incentivos, facilidad de uso |
| Manipulación digital | Auditorías, código abierto |
| Faccionalismo | Rotación, límite mandatos |
| Ingobernabilidad | Ejecutiva con autonomía operativa |
