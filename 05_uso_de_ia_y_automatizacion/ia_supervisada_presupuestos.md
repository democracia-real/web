# IA Supervisada en Presupuestos y Licitaciones

## Resumen

Uso de **modelos de IA** para **detectar desviaciones** en licitaciones y ejecución presupuestaria **en tiempo real**: patrones de colusión, adjudicaciones a medida, sobrecostes y anomalías que requieran supervisión humana.

## Objetivo

- **Monitorización continua** de contratos públicos y presupuestos.
- **Alertas** cuando se detecten patrones anómalos (mismas empresas ganando, requisitos a medida, desviaciones de precio).
- **IA como herramienta**, no como decisor: un humano valida cada alerta antes de cualquier actuación.

## Referencia

- Sistema **SALER** (Agencia Valenciana Antifraude): caso real de auditoría algorítmica.
- Ver [auditoria_algoritmica.md](auditoria_algoritmica.md) para el marco general.

## Dimensiones

| Área | Aplicación |
|------|------------|
| Licitaciones | Detección de colusión, requisitos a medida, reparto de lotes. |
| Ejecución | Desviaciones presupuestarias, facturas anómalas, plazos. |
| Cruce de datos | Conflictos de interés (funcionario–adjudicatario). |
| Supervisión | Comisión o unidad que recibe alertas y decide actuación. |
