Cómo Contribuir

Este es un proyecto abierto y colaborativo. Tu aportación es bienvenida.

Tipos de contribución

Tipo Descripción Habilidades
Investigación Aportar datos, estudios, análisis con fuentes Búsqueda, análisis
Documentación Mejorar o ampliar contenido existente Redacción, Markdown
Revisión Detectar errores, inconsistencias, mejoras Atención al detalle
Propuestas Plantear soluciones a problemas identificados Conocimiento del tema
Código Mejoras a la web o herramientas HTML, CSS, JS
Difusión Compartir el proyecto Redes, comunicación

Principios de colaboración

  • Objetividad — Análisis basado en hechos, no en ideología
  • Rigor — Citar fuentes, reconocer limitaciones
  • Constructividad — Críticas acompañadas de alternativas
  • Respeto — Debate de ideas, no ataques personales

Proceso de contribución

Para cambios en documentación

  1. Fork del repositorio en GitHub
  2. Crea una rama para tu cambio (git checkout -b mejora-seccion-x)
  3. Realiza los cambios siguiendo el formato existente
  4. Commit con mensaje descriptivo
  5. Push a tu fork
  6. Abre un Pull Request explicando el cambio

Para propuestas o investigación

  1. Abre un Issue describiendo la propuesta
  2. Discusión con la comunidad
  3. Desarrollo si hay consenso
  4. Pull Request con el contenido

Formato de documentos

Todos los documentos deben seguir esta estructura:

# Título

## Resumen
Breve descripción del contenido.

## Análisis
Desarrollo del tema con datos y fuentes.

## Propuesta (si aplica)
Solución concreta.

## Fuentes
- [Nombre](URL)
- [Nombre](URL)

Criterios de aceptación

  • Contenido alineado con los principios del proyecto
  • Fuentes citadas para afirmaciones de hecho
  • Formato consistente con el resto
  • Sin contenido ideológico partidista
  • Lenguaje claro y accesible

Código de conducta

  • Respeto a todos los participantes
  • Debate de ideas, no de personas
  • Aceptar críticas constructivas
  • Reconocer errores y corregir
  • Cero tolerancia a discriminación
Si compartes el diagnóstico y quieres aportar soluciones, este es tu proyecto. Bienvenido/a.
