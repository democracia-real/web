# Planificación a Largo Plazo: Visión de Estado

## Objetivo

Establecer mecanismos que permitan **políticas de Estado a 10-20 años**, blindadas frente a la volatilidad electoral, para sectores estratégicos donde la continuidad es esencial.

## El Problema Actual

### Volatilidad Legislativa

| Fenómeno | Descripción |
|----------|-------------|
| Ciclos de 4 años | Cada legislatura deshace lo anterior |
| Leyes "bandera" | Más simbólicas que efectivas |
| Reformas incompletas | Sin tiempo para implementar antes del cambio |
| Falta de evaluación | No se mide impacto antes de cambiar |

### Consecuencias

| Área | Impacto |
|------|---------|
| Educación | Múltiples leyes, ninguna consolidada |
| Energía | Cambios de modelo, inseguridad inversora |
| Pensiones | Reformas parciales, incertidumbre |
| Infraestructuras | Proyectos abandonados o cambiados |
| I+D | Presupuestos variables, fuga de talento |

## Modelo de Referencia: Planificación Estratégica

### El Caso Chino (Referencia, no Modelo)

China utiliza **Planes Quinquenales y Decenales**:

| Característica | Descripción |
|----------------|-------------|
| Horizonte | 5-15 años |
| Prioridades | Claras y medibles |
| Continuidad | Independiente de cambios de liderazgo |
| Revisión | Cada 5 años, ajuste sin abandono |
| Resultados | Consolidación industrial, tecnológica |

**Limitación:** Sistema autocrático, no aplicable como modelo político.

**Lección extraíble:** La continuidad en políticas estratégicas produce resultados.

## Propuesta: Arquitectura de la Continuidad

### Principio Fundamental

> *"Ciertos sectores estratégicos deben estar protegidos de la volatilidad partidista mediante mecanismos de consenso amplio y revisión técnica."*

### Sectores Estratégicos Propuestos

| Sector | Justificación |
|--------|---------------|
| **Educación** | Generaciones enteras afectadas por cambios |
| **Energía** | Inversiones a 20-30 años |
| **Pensiones** | Compromisos intergeneracionales |
| **Defensa** | Capacidades que tardan décadas en construir |
| **I+D** | Resultados a largo plazo |
| **Infraestructuras** | Proyectos de décadas |
| **Demografía** | Políticas con efecto a 20+ años |

### Mecanismos de Protección

#### 1. Pactos de Estado Reforzados

| Elemento | Propuesta |
|----------|-----------|
| Definición | Acuerdos parlamentarios con mayoría reforzada (3/5) |
| Contenido | Principios y objetivos a 10-20 años |
| Modificación | Solo por la misma mayoría reforzada |
| Evaluación | Cada 5 años por comisión técnica independiente |
| Vinculación | Presupuestos mínimos garantizados |

#### 2. Agencias Independientes

| Elemento | Propuesta |
|----------|-----------|
| Modelo | Autoridades administrativas independientes |
| Nombramiento | Por concurso, mandatos largos no renovables |
| Función | Ejecutar políticas del Pacto de Estado |
| Rendición | Al Parlamento, no al Gobierno |
| Ejemplo | Autoridad de Transición Energética |

#### 3. Presupuestos Plurianuales

| Elemento | Propuesta |
|----------|-----------|
| Horizonte | Marcos de gasto a 4-6 años |
| Sectores protegidos | Mínimos garantizados |
| Modificación | Mayoría reforzada |
| Supervisión | Autoridad fiscal independiente (AIReF ampliada) |

#### 4. Evaluación Obligatoria de Políticas

| Elemento | Propuesta |
|----------|-----------|
| Evaluación ex-ante | Impacto esperado antes de aprobar |
| Evaluación ex-post | Resultados reales tras implementación |
| Órgano | Agencia independiente de evaluación |
| Publicidad | Resultados públicos y vinculantes |
| Consecuencia | No se puede derogar sin evaluar |

### El Papel de la IA en la Planificación

| Aplicación | Función |
|------------|---------|
| Simulación de escenarios | Proyectar impacto de políticas a largo plazo |
| Monitorización de indicadores | Seguimiento en tiempo real de objetivos |
| Detección temprana | Alertas si hay desviaciones |
| Análisis de coherencia | Verificar que políticas no se contradigan |
| Benchmarking | Comparar con otros países automáticamente |

## Requisitos Legales

| Mecanismo | Tipo de Norma |
|-----------|---------------|
| Pactos de Estado | Resoluciones parlamentarias + leyes marco |
| Agencias independientes | Leyes orgánicas específicas |
| Presupuestos plurianuales | Reforma LGP + posible CE |
| Evaluación obligatoria | Ley ordinaria |

## Riesgos y Mitigaciones

| Riesgo | Mitigación |
|--------|------------|
| Déficit democrático | Revisión periódica con debate parlamentario |
| Rigidez excesiva | Mecanismos de ajuste con mayoría reforzada |
| Tecnocracia | Supervisión parlamentaria, transparencia |
| Captura por intereses | Rotación, incompatibilidades, publicidad |

## Indicadores de Éxito

| Indicador | Medición |
|-----------|----------|
| Continuidad legislativa | % de leyes estratégicas que sobreviven cambio de gobierno |
| Cumplimiento de objetivos | Evaluación de Pactos de Estado |
| Confianza inversora | Inversión extranjera en sectores estratégicos |
| Posición internacional | Rankings de competitividad, innovación |

## Hoja de Ruta

### Fase 1: Pilotos
1. Proponer un Pacto de Estado en un sector (ej. Energía)
2. Crear una agencia independiente piloto
3. Establecer evaluación obligatoria para nuevas leyes

### Fase 2: Consolidación
1. Extender modelo a otros sectores
2. Reformar presupuestos hacia plurianualidad
3. Fortalecer agencias existentes (AIReF, CNMC)

### Fase 3: Constitucionalización
1. Incluir principios de continuidad en CE
2. Proteger sectores estratégicos constitucionalmente
3. Crear órgano constitucional de evaluación
