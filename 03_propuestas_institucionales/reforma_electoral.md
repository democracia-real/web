# Reforma Electoral: Hacia una Representación Real

## Objetivo

Transformar el sistema electoral español para que los ciudadanos elijan verdaderamente a sus representantes y estos respondan ante sus electores, no ante las cúpulas de los partidos.

## Diagnóstico del Sistema Actual

| Problema | Consecuencia |
|----------|--------------|
| Listas cerradas y bloqueadas | No se elige a personas, solo a partidos |
| Circunscripciones provinciales | Diputados sin territorio identificable |
| Fórmula D'Hondt en distritos pequeños | Desproporcionalidad |
| Sin revocación | Impunidad durante 4 años |

## Propuestas de Reforma

### Opción 1: Sistema Electoral Mixto (Recomendada)

**Modelo:** Inspirado en el sistema alemán, adaptado a España.

#### Diseño Propuesto

| Componente | Detalle |
|------------|---------|
| **Escaños uninominales** | 175 diputados elegidos en distritos |
| **Escaños proporcionales** | 175 diputados por lista (nacional o autonómica) |
| **Doble voto** | Cada elector vota candidato + partido |
| **Barrera** | 3% nacional para escaños proporcionales |
| **Compensación** | Escaños adicionales si hay desproporción |

#### Definición de Distritos Uninominales

| Criterio | Detalle |
|----------|---------|
| Población | Similar en todos los distritos (~260.000 habitantes) |
| Contiguidad | Territorios conectados |
| Respeto a provincias | No partir provincias (si posible) |
| Revisión | Cada 10 años según censo |
| Órgano | Comisión independiente (no parlamento) |

#### Mecanismo de Elección

```
CIUDADANO tiene 2 VOTOS:

VOTO 1 (Uninominal):
- Elige a UN candidato de su distrito
- Gana el más votado (mayoría simple)
- Diputado identificable y responsable

VOTO 2 (Lista):
- Elige a UN partido
- Determina la composición proporcional del Congreso
- Lista preferiblemente desbloqueada
```

#### Ejemplo de Funcionamiento

```
Partido A:
- Gana 50 distritos uninominales = 50 escaños
- Obtiene 30% del voto de lista = 105 escaños (30% de 350)
- Ya tiene 50, le corresponden 55 más de la lista

Partido B:
- Gana 10 distritos = 10 escaños
- Obtiene 25% del voto = 87,5 ≈ 88 escaños
- Ya tiene 10, le corresponden 78 de la lista

Resultado: Proporcionalidad global + representación territorial
```

### Opción 2: Listas Desbloqueadas (Reforma Mínima)

**Modelo:** Mantener el sistema actual pero permitir voto preferencial.

| Cambio | Detalle |
|--------|---------|
| Voto preferencial | El elector puede marcar candidatos dentro de la lista |
| Reordenación | Candidatos con más votos suben en el orden |
| Umbral | Se requiere X% de votos preferenciales para alterar orden |

**Ventaja:** No requiere reforma constitucional ni redibujar distritos.

**Limitación:** No crea diputados territorialmente identificables.

### Opción 3: Sistema Uninominal Puro

**Modelo:** 350 distritos, cada uno elige 1 diputado.

| Aspecto | Detalle |
|---------|---------|
| Distritos | 350, de similar población |
| Sistema | Mayoritario a dos vueltas |
| Primera vuelta | Todos los candidatos |
| Segunda vuelta | Dos más votados (si ninguno >50%) |

**Ventaja:** Máxima conexión representante-territorio.

**Inconveniente:** Pérdida de proporcionalidad.

## Elementos Complementarios (Cualquier Opción)

### Revocación del Mandato

| Elemento | Propuesta |
|----------|-----------|
| Iniciativa | X% del censo del distrito |
| Causas | Incumplimiento de compromisos, conducta |
| Procedimiento | Referéndum de revocación |
| Protección | Período mínimo antes de poder revocar |

### Democracia Interna en Partidos

| Elemento | Propuesta |
|----------|-----------|
| Primarias obligatorias | Para seleccionar candidatos |
| Transparencia | Financiación y decisiones internas |
| Derechos de afiliados | Voz y voto real |
| Sanciones | Por incumplimiento |

### Limitación de Mandatos

| Cargo | Límite Propuesto |
|-------|-----------------|
| Diputado | Máximo 3 legislaturas (12 años) |
| Senador | Máximo 3 legislaturas |
| Presidente | Máximo 2 mandatos (8 años) |

### Para Listas Desbloqueadas

| Reforma | Tipo | Dificultad |
|---------|------|------------|
| LOREG | Ley Orgánica | Media |

## Comparativa de Opciones

| Criterio | Mixto | Desbloqueadas | Uninominal |
|----------|-------|---------------|------------|
| Proporcionalidad | Alta | Alta | Baja |
| Territorialidad | Alta | Baja | Muy alta |
| Rendición cuentas | Alta | Media | Muy alta |
| Viabilidad política | Baja | Media | Muy baja |
| Complejidad técnica | Alta | Baja | Media |

## Riesgos y Mitigaciones

| Riesgo | Mitigación |
|--------|------------|
| Gerrymandering (manipulación distritos) | Comisión independiente de delimitación |
| Fragmentación excesiva | Barrera del 3% nacional |
| Candidatos personalistas | Límite de mandatos |
| Resistencia de partidos | Presión ciudadana, ejemplos internacionales |

## Hoja de Ruta

### Corto Plazo (1-2 años)
1. Campaña de concienciación ciudadana
2. Propuesta de listas desbloqueadas
3. Debate parlamentario

### Medio Plazo (3-5 años)
1. Estudio técnico de distritos
2. Propuesta de sistema mixto
3. Búsqueda de consenso parlamentario

### Largo Plazo (5-10 años)
1. Reforma de la LOREG
2. Primera elección con nuevo sistema
3. Evaluación y ajustes
