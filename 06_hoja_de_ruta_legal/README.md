# 06 - Hoja de Ruta Legal para Crear el Partido

## Resumen Ejecutivo

Este módulo detalla los **pasos legales y procedimentales** necesarios para constituir e inscribir un partido político en España, conforme a la Ley Orgánica 6/2002 de Partidos Políticos.

## Requisitos Esenciales

### Promotores

| Requisito | Detalle |
|-----------|---------|
| Número mínimo | 3 personas físicas |
| Edad | Mayores de edad |
| Capacidad | Pleno ejercicio de derechos |
| Nacionalidad | Española (no exigida expresamente, pero implícita) |
| Antecedentes | Sin condena por asociación ilícita o delitos graves (Títulos XXI-XXIV CP) |

### Documentación Necesaria

| Documento | Requisitos |
|-----------|------------|
| **1. Solicitud de inscripción** | Dirigida al Registro, firmada por promotor |
| **2. Acta fundacional** | En escritura pública (notario) |
| **3. Estatutos** | Conforme a art. 3.2 LO 6/2002 |
| **4. DNI promotores** | Copia de los tres mínimo |

### Contenido del Acta Fundacional

| Elemento | Contenido |
|----------|-----------|
| Identificación promotores | Nombre, apellidos, DNI, domicilio |
| Denominación del partido | Nombre completo y siglas |
| Órganos directivos provisionales | Nombres y cargos |
| Domicilio del partido | Dirección física |
| Estatutos | Texto íntegro |

### Contenido Obligatorio de Estatutos (Art. 3.2 LO 6/2002)

| Apartado | Contenido Requerido |
|----------|---------------------|
| a) | Denominación, siglas, símbolo, domicilio, ámbito |
| b) | Requisitos y procedimiento de afiliación |
| c) | Derechos y deberes de afiliados |
| d) | Órganos de gobierno: composición, convocatoria, atribuciones |
| e) | Procedimientos de elección de órganos |
| f) | Régimen disciplinario |
| g) | Procedimiento de modificación de estatutos |
| h) | Régimen de responsabilidad económico-financiera |
| i) | Causas de disolución y destino del patrimonio |

### Requisitos Adicionales (desde 2015)

| Requisito | Norma |
|-----------|-------|
| Sistema de prevención de conductas ilícitas | Art. 3.2 LO 6/2002 (reforma) |
| Responsable de gestión económico-financiera | LO 3/2015 |

## Procedimiento de Inscripción

### Fases del Proceso

```
1. CONSTITUCIÓN
   - Reunión de promotores
   - Redacción de estatutos
   - Formalización ante notario

2. SOLICITUD
   - Presentación de documentación
   - Entrada en Registro de Partidos

3. EXAMEN
   - Verificación de requisitos formales
   - Posible subsanación de defectos

4. RESOLUCIÓN
   - Inscripción (o denegación)
   - Notificación al promotor

5. PUBLICACIÓN
   - Publicación en BOE
   - Adquisición de personalidad jurídica
```

### Plazos

| Fase | Plazo |
|------|-------|
| Examen de documentación | 20 días hábiles desde entrada |
| Subsanación de defectos | Suspende el plazo |
| Silencio administrativo | Positivo (inscripción automática) |

### Formas de Presentación

| Vía | Detalle |
|-----|---------|
| Presencial | Registro del Ministerio del Interior (Madrid) |
| Correo postal | A la misma dirección |
| Sede electrónica | www.interior.gob.es (con certificado digital) |
| Registros art. 16.4 Ley 39/2015 | Cualquier registro administrativo |

### Particularidades

| Aspecto | Detalle |
|---------|---------|
| Acta notarial | Debe ser ORIGINAL, no copia simple |
| Formato electrónico | Si es e-notarial, debe incluir CSV verificable |
| Depósito | El original queda en el Registro |

## Denominación del Partido

### Prohibiciones

| No puede | Razón |
|----------|-------|
| Inducir a error o confusión | Sobre la identidad |
| Coincidir con otro inscrito | Ni semejarse fonéticamente |
| Usar nombre de partido ilegalizado | Evitar continuidad |
| Usar nombres de personas físicas | Protección de identidad |
| Usar marcas registradas | Propiedad intelectual |

### Recomendación

Antes de elegir nombre:
1. Consultar el [listado de partidos inscritos](https://infoelectoral.interior.gob.es/es/formaciones-politicas/registro-de-partidos-politicos/acceso-al-listado-de-formaciones-politicas)
2. Verificar que no haya similitud fonética
3. Comprobar marcas en OEPM

## Después de la Inscripción

### Obligaciones Inmediatas

| Obligación | Detalle |
|------------|---------|
| Alta fiscal | CIF del partido |
| Cuenta bancaria | A nombre del partido |
| Responsable económico | Designación obligatoria |
| Libros contables | Según LO 3/2015 |

### Obligaciones Periódicas

| Obligación | Frecuencia |
|------------|------------|
| Cuentas anuales | Anual (al Tribunal de Cuentas) |
| Renovación de cargos | Según estatutos |
| Congresos/Asambleas | Según estatutos |

## Causas de No-Inscripción

| Causa | Consecuencia |
|-------|--------------|
| Defectos formales subsanables | Requerimiento de subsanación |
| Defectos formales no subsanados | Denegación |
| Denominación prohibida | Denegación |
| Indicios de ilicitud | Remisión al Ministerio Fiscal |

## Extinción del Partido

### Causas de Extinción

| Causa | Procedimiento |
|-------|---------------|
| Disolución voluntaria | Asamblea y comunicación al Registro |
| Fusión con otro | Inscripción del nuevo partido |
| Declaración judicial de ilegalidad | Sentencia firme |
| No adaptación de estatutos | Procedimiento de extinción |
| No renovación de cargos | Procedimiento de extinción |
| No presentación de cuentas | Procedimiento de extinción |

## Cronograma Tipo

### Mes 1: Preparación

| Semana | Actividad |
|--------|-----------|
| 1-2 | Identificar y reunir promotores (mínimo 3) |
| 2-3 | Redactar borrador de estatutos |
| 3-4 | Elegir denominación (verificar disponibilidad) |

### Mes 2: Formalización

| Semana | Actividad |
|--------|-----------|
| 1 | Revisar estatutos con asesoría jurídica |
| 2 | Cita con notario |
| 3 | Firma del acta fundacional |
| 4 | Preparar documentación para Registro |

### Mes 3: Inscripción

| Semana | Actividad |
|--------|-----------|
| 1 | Presentar solicitud de inscripción |
| 2-4 | Esperar resolución (20 días hábiles) |
| 4 | Recibir inscripción |

### Mes 4: Puesta en Marcha

| Semana | Actividad |
|--------|-----------|
| 1 | Alta fiscal, cuenta bancaria |
| 2 | Web, comunicación |
| 3-4 | Primera asamblea de afiliados |
