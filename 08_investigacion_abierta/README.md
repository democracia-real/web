## Principios de Investigación

### 1. Basada en Evidencia
- Datos antes que opiniones
- Fuentes verificables
- Metodología explícita
- Resultados reproducibles

### 2. Abierta y Colaborativa
- Cualquiera puede contribuir
- Revisión por pares
- Publicación de resultados
- Reconocimiento de contribuciones

### 3. Orientada a la Acción
- Investigar para resolver, no solo para saber
- Priorizar lo que tiene impacto
- Convertir hallazgos en propuestas

### 4. Humilde y Corregible
- Reconocer limitaciones
- Aceptar críticas
- Actualizar cuando hay nueva evidencia

## Metodología de Investigación

### Tipos de Investigación

| Tipo | Descripción | Ejemplo |
|------|-------------|---------|
| **Revisión documental** | Análisis de fuentes existentes | Leyes, informes, estadísticas |
| **Análisis de datos** | Procesamiento de datos públicos | Open data, APIs oficiales |
| **Entrevistas** | Conversaciones con expertos | Académicos, funcionarios |
| **Casos de estudio** | Análisis en profundidad | Proyectos europeos, reformas |
| **Comparativas** | Análisis entre países/regiones | Sistemas electorales |

### Proceso Estándar

```
1. DEFINIR PREGUNTA
   - Concreta y respondible
   - Relevante para el proyecto
   
2. REVISAR LITERATURA
   - ¿Qué se sabe ya?
   - ¿Qué fuentes existen?
   
3. DISEÑAR METODOLOGÍA
   - Cómo vamos a investigar
   - Qué datos necesitamos
   
4. RECOPILAR DATOS
   - Fuentes primarias y secundarias
   - Documentar origen
   
5. ANALIZAR
   - Procesar información
   - Identificar patrones
   
6. VALIDAR
   - Revisión por pares
   - Contraste con expertos
   
7. PUBLICAR
   - Documento con hallazgos
   - Datos abiertos si aplica
   
8. TRADUCIR A PROPUESTA
   - ¿Qué implica para el programa?
   - ¿Qué acción se deriva?
```

### Estándares de Calidad

| Criterio | Requisito |
|----------|-----------|
| Fuentes | Citadas y verificables |
| Metodología | Explícita y reproducible |
| Datos | Disponibles para verificación |
| Revisión | Al menos 2 revisores |
| Actualización | Fecha de última revisión |
| Limitaciones | Reconocidas explícitamente |

## Colaboración

### Cómo Contribuir

| Forma | Descripción |
|-------|-------------|
| Proponer pregunta | Sugerir temas de investigación |
| Aportar fuentes | Compartir datos o referencias |
| Investigar | Realizar análisis y documentar |
| Revisar | Validar investigaciones de otros |
| Difundir | Compartir hallazgos |

### Proceso de Contribución

```
1. Fork del repositorio
2. Crear documento o modificar existente
3. Seguir formato y estándares
4. Pull Request con descripción
5. Revisión por mantenedores
6. Merge si cumple criterios
7. Reconocimiento en colaboradores
```

### Reconocimiento

| Contribución | Reconocimiento |

| Investigación original | Autoría en documento |
| Revisión | Mención como revisor |

## Preguntas Abiertas Actuales

### Alta Prioridad

1. ¿Cuál es el coste real de la corrupción en España?
2. ¿Cuántos días/euros cuesta abrir una empresa vs. media UE?
3. ¿Qué porcentaje de fondos europeos genera resultados medibles?
4. ¿Cuál es el tiempo medio de resolución judicial por tipo de caso?
5. ¿Cuántas políticas de una legislatura son revertidas por la siguiente?

### Media Prioridad

6. ¿Qué sistemas de democracia líquida se han implementado y con qué resultados?
7. ¿Cuál es el impacto de la cuota de autónomos en el emprendimiento?
8. ¿Qué países usan IA en administración pública y qué resultados obtienen?
9. ¿Cuántos funcionarios hay por habitante en España vs. UE?
10. ¿Cuál es la tasa de abstención por perfil sociodemográfico?

### Para Explorar

11. ¿Cómo medir la "calidad democrática" de un sistema?
12. ¿Qué reformas han conseguido reducir la corrupción en otros países?
13. ¿Cuál es el impacto económico de la burocracia urbanística?
14. ¿Cómo afecta el sistema electoral a la representación de minorías?
15. ¿Qué mecanismos de rendición de cuentas funcionan mejor?
